자바스크립트가 없던 시절의 브라우저
→ 대부분 파일기반 라우팅* 기반으로 특정 경로에 저장된 정적 HTML 조회/파싱 수행
* 파일기반 라우팅: URL과 파일 경로를 매핑해 페이지를 랜더링하는 방식
개발과 유지보수에 편리하나 필수는 아님 → 요청(URL)과 실제 정적자원 경로는 라우팅 전략에 따라 불일치하는 경우도 많다.
→ 여기서, 템플릿 엔진으로 HTML에 데이터 패치해 동적 콘텐츠를 생성하는 방식으로 발전
템플릿 엔진 예: JSP, Thymeleaf, Mustache, Handlebars, Pug...
